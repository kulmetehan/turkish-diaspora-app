# N4.1 – News List View (Voor jou)

## Overzicht
- Route: `/news`
- Doelgroep: mobiele en desktop gebruikers
- Feed: `feed=diaspora` (“Voor jou”)
- Componenten: `useNewsFeed`, `NewsCard`, `NewsList`, `FooterTabs`

## UI
- Mobiel-first layout met titel “Nieuws voor jou” + korte beschrijving.
- Nieuwsitems tonen titel, snippet, bron, datum, tags en optioneel thumbnail.
- Kaarten zijn klikbaar en openen de originele URL in een nieuw tabblad.
- Loading/error/empty states volgen dezelfde kaartstijl als `LocationList`.

## Dataflow
1. `useNewsFeed({ feed: "diaspora", pageSize: 20 })`
   - gebruikt `fetchNews` → `GET /api/v1/news?feed=diaspora&limit=20&offset=…`
   - beheert `items`, `hasMore`, `isLoading`, `isLoadingMore`, `error`
   - levert `reload()` en `loadMore()` helpers
2. `NewsList`
   - toont skeletons tijdens initial load
   - toont foutkaart + “Opnieuw proberen” bij error
   - toont lege kaart wanneer er nog geen resultaten zijn
   - voegt “Meer laden” knop toe zolang `hasMore = true`

## Paginatie
- Offset-based; standaard `limit=20`.
- `loadMore` verhoogt offset met `pageSize` en voegt resultaten toe.
- “Meer laden” verdwijnt wanneer alle items geladen zijn.

## Hergebruikte patronen
- Card/skeleton tokens uit `components/ui/card` en `components/ui/skeleton`.
- Turkspot brand kleuren via `index.css` + Tailwind tokens (`bg-card`, `bg-brand-redSoft`).
- FooterTabs bevat al de “News” tab; geen extra navigatie nodig.

## Testing
- `NewsCard.spec.tsx`: controleert rendering + clickgedrag.
- `NewsPage.spec.tsx`: mocked hook → smoke test + load-more interactie.
